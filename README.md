# Project oauth2-example

## Requisitos

- python3

## Executando

- Instale as dependencias
```
pip install Flask requests-oauthlib
```
- Altere o arquivo app.py, informando os dados de autentificacao
  - app.secret_key
  - client_id
  - client_secret
- Rode a aplicação
```
python app.py
``` 
- Acesse a url de Login
```
http://127.0.0.1:5000/
```
