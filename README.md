# Project oauth2-example

## Requisitos

- python3

## Executando

- Instale as dependencias
```
pip install Flask requests-oauthlib
```
- Rode a aplicação
  - Lembre-se de ajustar os parametros conforme suas informacoes
```
python app.py \
  --secret_key 'X' \
  --client_id 'Y' \
  --client_secret 'Z'
``` 
- Acesse a url de Login
```
http://127.0.0.1:5000/
```
